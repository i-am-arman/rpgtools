Arman's RPG Tools

This site holds various tools I've whipped up for the RPGs I've played, as either a GM or a player (or both!). Hopefully someone can find them useful.

The ACKS Tools are designed for use with the Adventurer, Conqueror, King system, using the guide, player's guide, and any additional publications I can get my hands on.

The Shifting Horizon tools are actually slightly out of date; they are tied to the blog, which is somewhat more likely to be in better shape.
